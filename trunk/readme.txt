== Description ==
Markeing toolbar is a new way of wordpress marketing. Using marketing toolbar, you can show the advertisement in your blog/site. It provides you to hide the toolbar in some pages also if you need. Auto contract and expand feature leads the good interaction of toolbar with visitors. Once user mouseover  it, it stops auto vibration. Though it also stop auto vibration after certain time adjustable by admin and number of times of vibration,gap between two vibrations can be adjust from admin. It provides you the facility to use option form codes like aweber in very efficient way.

== Installation ==

Its very easy to install the plugin like other simple plugin you do.

== Frequently Asked Questions ==

= Close button is not showing in my toolbar at front end =

Please check whether you have selected display close button to yes in admin

= How do i hide my toolbar in front page =

In admin, there is exclude post. Enter page/post id of pages separated by comma(,) where you like to hide the marketing toobar plugin
